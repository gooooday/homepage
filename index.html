Dalian DHC System Mail Service

This application is an internal system tool used to send automated notification emails using the Gmail API.

Service Description

The purpose of this application is to send system-generated emails such as:

  • Task execution results
  • System status notifications
  • Operational alerts

All emails are sent only based on explicit user actions or automated tasks configured by the user. This application does NOT read, store, or modify any user emails.

Gmail API Usage

This application only uses the following Gmail API scope:

  • https://www.googleapis.com/auth/gmail.send

The Gmail API is used strictly for sending emails. No email reading, deletion, or modification is performed.
