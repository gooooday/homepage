Privacy Policy

This application only uses the Gmail API scope gmail.send to send emails on behalf of the authenticated user.

Information We Collect

We do NOT collect, read, store, modify, or delete any user emails.

We do NOT store any Gmail content, attachments, or metadata.

How We Use Information

The Gmail API is used only to send system-generated notification emails explicitly triggered by the user.

Data Sharing

We do NOT share any user data with any third party.

We do NOT use any data for advertising, marketing, or tracking purposes.

Data Security

All access tokens and refresh tokens are securely stored and used only for authorized Gmail sending operations.

User Control

Users can revoke access to this application at any time through their Google Account security settings.
